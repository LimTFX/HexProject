# 🔐 Шифровальщик текстовых сообщений

## Описание проекта
Веб-приложение для шифрования и дешифрования текста с использованием классических алгоритмов:
- **Шифр Цезаря** (с возможностью задания сдвига)
- **Шифр Атбаш** (зеркальное отображение алфавита)

## Особенности
- ✅ Поддержка **русского (кириллица, 33 буквы)** и **английского (латиница, 26 букв)** алфавитов
- ✅ Автоматическое определение языка текста
- ✅ Сохранение результата в `.txt` файл
- ✅ Загрузка файла для расшифровки
- ✅ Современный адаптивный интерфейс
- ✅ Полностью клиентская работа (без сервера)

## Технологии
- HTML5
- CSS3 (Flexbox, анимации, адаптив)
- JavaScript ES6+ (массивы, объекты, строки, File API, Blob)

## Установка и запуск
1. Скачайте все файлы:
   - `index.html`
   - `style.css`
   - `script.js`
2. Поместите их в одну папку
3. Откройте `index.html` в любом современном браузере (Chrome, Firefox, Edge, Safari)

(Так же можно скачать сразу zip-архив или клонировать к себе в дирректорию)

## Использование
1. Введите текст в поле ввода (можно на русском или английском)
2. Выберите алгоритм: Цезарь или Атбаш
3. Для Цезаря укажите сдвиг (1–32 для русского языка, 1-25 для английского языка)
4. Выберите режим: Зашифровать или Расшифровать
5. Нажмите «Выполнить»
6. Результат появится в области вывода
7. Сохраните результат в файл или загрузите файл для расшифровки

## Примеры
| Исходный текст | Алгоритм | Сдвиг | Результат |
|----------------|----------|-------|-----------|
| Привет | Цезарь | 7 | Цчпилщ |
| Hello | Цезарь | 7 | Olssv |
| HelloПривет123456 | Цезарь | 11 | SpwwzЪыумпэ123456|
| AbCdEf | Атбаш | - | ZyXwVu |
| аБвГдЕ | Атбаш | - | яЮэЬыЪ |
| HelloПривет123456 | Атбаш | - | SvoolПоцэъм123456 |
